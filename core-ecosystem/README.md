# Core Ecosystem

Guide per l'installazione dell'infrastruttura Docker principale.

## Ordine di Installazione

Seguire le guide nell'ordine indicato dai prefissi numerici:

1. **[01-docker-portainer.md](01-docker-portainer.md)** - Installazione Docker e Portainer per la gestione container
2. **[02-docker-network.md](02-docker-network.md)** - Creazione della rete Docker condivisa tra i servizi
3. **[03-glpi-installation.md](03-glpi-installation.md)** - Installazione GLPI con MariaDB per la gestione IT
4. **[04-n8n-installation.md](04-n8n-installation.md)** - Installazione n8n con PostgreSQL per l'automazione

## Prerequisiti

- Sistema operativo Linux (consigliato Ubuntu/Debian)
- Accesso root o utente con privilegi sudo
- Connessione internet per il download delle immagini Docker
